# Rock Paper Scissors Game
#Description

A simple Rock-Paper-Scissors game built with HTML, CSS, and JavaScript.
The user can select rock, paper, or scissors, and the computer randomly selects its move. The game displays the winner, updates the score, and allows continuous play.

# Features

Interactive UI with clickable choices.

Scoreboard to track user vs computer scores.

# Winner message with color indication:

Green → You win

Red → You lose

Blue/Default → Draw

Responsive design (works on different screen sizes).

# Technologies Used

HTML – for structure of the game.

CSS – for styling the buttons, scoreboard, and messages.

JavaScript – for game logic, score updates, and event handling.

# How to Run

https://github.com/Archana-Lohana/Rock-Paper-Scissor

Open index.html in a web browser.

Click on Rock, Paper, or Scissors to play.
